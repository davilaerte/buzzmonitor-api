# buzzmonitor-api

To run the project, run the command:
```
mvn spring-boot:run
```
To connect to the database ([ElasticSearch](https://www.elastic.co/products/elasticsearch)):

1. Run the ElasticSearch in **localhost:9300** (Default port)
